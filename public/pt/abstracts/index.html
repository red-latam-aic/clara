Palestrantes convidados

Os palestrantes convidados são pesquisadores de destaque de origem latino-americana, especializados na aplicação de metodologias de IA nas ciências naturais e na engenharia.

  • Prof. Fernanda Duarte, Oxford University, United Kingdom. Topic: Machine Learning in Chemistry.
  • Prof. Fernanda Selingardi Matias, Universidade Federal de Alagoas, Brasil. Topic: Machine Learning for Neurosciences.
  • Dr. Oscar Mendez-Lucio, Recursion, Spain. Topic: Machine Learning for Drug Design, applications of ML in industrial settings.
  • Prof. Vinicius Mikuni, Nagoya University, Japan. Topic: Machine Learning in High-Energy Physics.
  • Anais Möller, Swinburne University, Australia. Topic: Machine Learning in Astronomy.
  • Ehecatl Antonio del Río Chanona, Imperial College London, United Kingdom. Topic: Machine Learning in Bioprocess Engineering.

Organizadores

  • Dr. Juan Carlos San Vicente, Temple University, USA. Topic: Machine Learning for excited state chemistry.
  • Dr. Manuel Szewc, Universidad de San Martín, Argentina. Topic: Machine Learning in HEP.
  • Dr. Luis Itza Vázquez-Salazar, Heidelberg University, Germany. Topic: Generative Machine Learning in Chemistry.
